Who do I talk to if I have Questions?

The best place to receive assistance is in our support forums. If you have a private matter to discus simply complete the form on our contact page. We willrespond promptly.

How can I download Styleflow library?

To download styleflow library, sign up to styleflow website or sign in if you are already registered. Then go to the library page and download the library zip files from the library page.

How can I use basic animations on HTML elements?

To use our animations on your projects, you will have to add the library to the same folder’s directory. Then add the name of the library file “styleflow.css” to the link tag of your html head. After this, you can then include the different styles to the class of tags you wish to animate.

Can I add intensity to my animations?

Yes! you can add intensity to your animations by adjusting the animations to suit your satisfaction.

How can I report when someone has plagiarised my work and uploaded it?

If someone is claiming your work is their own on StyleFlow, please report the animation and provide a link to the original animation so we can verify it.
